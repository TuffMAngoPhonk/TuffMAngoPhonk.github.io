# 🦖 Offline Dino Game

A simple Chrome Dino-style game built using HTML, CSS, and JavaScript.

## 🎮 Features
- Jumping dino
- Random cactus obstacles
- Score system
- Simple controls

## 🚀 How to Run
Just open `index.html` in your browser.

## 🎯 Controls
- Click Jump button / tap screen → Jump
- Avoid obstacles

## 📁 Tech
- HTML5 Canvas
- Vanilla JavaScript
- CSS
